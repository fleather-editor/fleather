# Fleather
*Soft and gentle rich text editing for Flutter applications based on [Zefyr](https://github.com/memspace/zefyr).*

## Features
* Inline attributes like **bold**, *italic*, ~~strikethrough~~ and etc.
* Line attributes like direction, alignment, heading, number and bullet list and etc.
* Block attributes like code, quote and etc.
* Markdown-inspired semantics
* Support for inline and block embeds
* Support for markdown shortcuts
* Using [Quill.js Delta](https://quilljs.com/docs/delta) as underlying data format by [Parchment](packages/parchment/README.md), Fleather is ready for collaborative editing using [OT](https://en.wikipedia.org/wiki/Operational_transformation) or [CRDT](https://en.wikipedia.org/wiki/Conflict-free_replicated_data_type) (Not provided as built-in functionality yet)
